# Tacokit Sandbox

The Tacokit Sandbox, hosted on Heroku, is useful for demonstrating the
OAuth flow with Trello. It also provides an endpoint for testing Trello
webhook callbacks.

Visit [Tacokit Sandbox](https://aqueous-depths-5917.herokuapp.com)
